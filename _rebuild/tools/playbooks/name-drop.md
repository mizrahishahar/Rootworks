# Name-drop

A modifier, not a campaign: drop a relevant senior colleague's name to lift the reply. Lands in structured orgs where the org chart is real; skip it in tiny or messy companies where the colleague is a guess.

## The play

- **A conditional line or PS** — "wasn't sure if this sits with you or {{colleague_name}}, whoever owns {{area}}." It signals you did homework and gives them an easy forward.
- **Skip the line entirely when no colleague is found** — a blank or wrong name is worse than nothing.
- **Always test with-vs-without** — it is a lift, not a certainty, and it costs a data layer.
- Everything else follows the base playbook it modifies.
