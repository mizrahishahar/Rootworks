# Trigify

## Description
Intent-based contact source: surfaces people from an engagement or intent signal across LinkedIn, social, and content. A standing automation runs the signal on its own cadence rather than inside a session.

## Query
An intent signal: the audience plus the trigger that surfaces them (a tracked profile, a topic, an engagement event).

## Claude actions (MCP)
Define the signal over the Trigify MCP: create a search, track profiles, or set a social-signal subscription.

## Owner actions
Stand up the signal automation and keep it running.

## To know
Not yet runnable end to end: how surfaced people become a claudeschema pull the handler can land is still open. Treat this as a definition until the first signal is built.

## Automations
None yet. When the standing signal is built, its per-client flow lands the surfaced people as a claudeschema pull.
