# Defer / not-now

"Check back in a month", "after the season." Honor it, date it, stay warm, short. Never push a call onto a clear not-now.

## When
A genuine timing deferral - not an objection in disguise. If it reads as a soft objection ("we're all set for now"), it is Conversation, not this.

## The play
One or two lines. Acknowledge the timing, name when you will return so it is on record, keep the door warm. That is the whole message.

## Never
Push a call onto a clear not-now; apologise; add a pitch; leave the return undated.

## Example
A small, medium-fit shop replied "check back in a month":

> No problem at all - I'll check back in a month.

Honors the timing, stays warm, no needy push, no apology. A clean hold that keeps the door open.
