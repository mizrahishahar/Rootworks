# Trigify

Intent-based contact source: surfaces people from an engagement or intent signal across LinkedIn, social, and content. A standing automation runs the signal on its own cadence rather than inside a session; surfaced people land continuously in the client Intent table in [[clayroots]], not in dated build tables.

## The query
An intent signal: the audience plus the trigger that surfaces them - a tracked profile, a topic, an engagement event.

## Actions (MCP)
Define the signal over the Trigify MCP: create a search, track profiles, or set a social-signal subscription.

## Operator actions
Stand up the per-client intent router in n8n and keep it running.

## Spend check
Subscription; the signal definition is the cost control.
